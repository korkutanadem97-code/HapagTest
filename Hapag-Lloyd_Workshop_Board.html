Dual-Product Orchestration

Three parallel streams — MIAW, Agentforce, Agile Backlog — orchestrated by a 6-person core team.

⏱ Block Timer
30:00

🔀 Three-Stream Delivery Model

STREAM 1 MIAW

  • Channel Provisioning & Routing
  • Agent Console Configuration
  • SLA Definitions per Market
  • Compliance Setup (GDPR etc.)
  • Staffing Models & Fallback

STREAM 2 Agentforce

  • Intent Recognition & NLP Training
  • Conversational Flow Design
  • Knowledge Base Integration
  • Handover Logic to Live Agents
  • Model Versioning & Rollback

STREAM 3 Agile Backlog

  • Feature Enhancements
  • Bug Fixes & Performance
  • Technical Debt Reduction
  • Ad-hoc Requests (15-20% Buffer)
  • PO-prioritized per Sprint

🏭 Industrialization Principles

♻ Reuse over Rebuild

Every MIAW config and every Agentforce flow as a reusable template. Market-specific adaptations are parameterized.

📐 80% Standard, 20% Local

Core flows standardized globally. Only language, cultural tone, and local compliance adapted per market.

⚡ Automate the Checklist

Pre-launch checklist progressively automated. Each new market requires less manual effort than the last.

📊 Measure & Iterate

KPI review at Day 3, Day 7, Day 14. Learnings are fed back into templates.

Country Rollout Standards

5 dimensions that every market must meet before go-live.

⏱ Block Timer
25:00

🌎 Language & Localization

  • Professional translation + native-speaker review
  • ≥200 utterances per intent in target language
  • Fallback flow for unsupported languages
  • Cultural tone guidelines (formal vs. informal etc.)

⚖ Compliance & Privacy

  • GDPR (EU) / LGPD (BR) / PIPL (CN)
  • Consent capture flows validated
  • Data residency technically implemented
  • Legal sign-off on automated responses

⚙ Technical Readiness

  • SF Messaging Channel provisioned
  • Agentforce ≥95% intent accuracy
  • CRM integration verified
  • Response time <3s, monitoring active

👥 Operational Readiness

  • Agent team trained
  • Staffing model validated (2-week ramp)
  • SLAs agreed (<60s first response)
  • Knowledge base in target language

🚦 Go-Live Criteria

≥80% readiness across all 5 dimensions.
No dimension below 60% (hard blocker).
Assessment by Governance Board.
